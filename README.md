# barcode-buddy-web-scan

A web-based barcode scanner that integrates with [Barcode Buddy](https://github.com/Forceu/barcodebuddy/).

## Building

Run `go build` to build the project.

## Environment Variables

- `BBUDDY_HOST`: The host URL for Barcode Buddy API
- `BBUDDY_API_KEY`: The API key for Barcode Buddy
- `PORT`: The port to run the server on (default: 8080)
